# Raj ERP

Raj ERP is a static HTML, CSS, and JavaScript frontend demo for a mens wear shop ERP and POS billing system.

## How to Run

Open `index.html` directly in any browser.

No npm, React, Vite, build command, or local server is required.

## Screens Included

- Login page
- Dashboard
- Product management
- POS billing screen
- Dark/light mode
- Mobile responsive layout
